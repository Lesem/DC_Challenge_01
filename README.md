# DC_Challenge_01
Análisis de eficiencia de ventas para Alura Store.

Este proyecto presenta un analisis de datos detallado para identificar cual de las cuatro tiendas del Sr. Juan presenta las mayores ineficiencias operativas, con el fin de recomendar su venta estrategica.

Hallazgos Principales
Identificacion de Ineficiencias:
A traves del analisis de KPIs, se determino que la Tienda 1, a pesar de tener los ingresos mas altos, es la unidad menos eficiente del grupo.

Costos Operativos:
La Tienda 1 registra el costo de envio promedio mas elevado, superando los 26,000, lo que reduce significativamente el margen de ganancia real en comparacion con las otras sedes.

Satisfaccion del Cliente:
Existe una correlacion negativa en la Tienda 1, donde el alto volumen de ventas coincide con la calificacion promedio mas baja (3.97), sugiriendo problemas en el servicio o calidad de entrega.

Comparativa de Eficiencia:
La Tienda 3 se identifica como el modelo operativo ideal, manteniendo ingresos competitivos con la calificacion mas alta (4.05) y costos de envio mas controlados.

Recomendacion Final
Se recomienda al Sr. Juan proceder con la venta de la Tienda 1.

Justificacion:
La decision se basa en que la Tienda 1 representa el mayor riesgo operativo y reputacional. Su alta facturacion no compensa los elevados costos de logistica y la baja satisfaccion del cliente. La venta de esta unidad permitira sanear el portafolio de negocios y concentrar recursos en las tiendas que demuestran mayor eficiencia y fidelidad de los clientes.
